# postgres-scratch
Scratchpad for postgres.

```bash
# Clone repository:
git clone git@github.com:Chumbaloo/postgres-scratch.git

# OR

git clone https://github.com/Chumbaloo/postgres-scratch.git


# build
make build

# run
make debug

# start postgres
root@<containerID>:/app# ./start.sh

postgres=\#  

```
